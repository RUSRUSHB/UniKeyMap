# UniKeyMap

跨平台统一键位方案：左 Ctrl/Alt 交换 + Caps Lock 导航层，在 Windows、Mac、Linux 上保持一致的输入与导航习惯。

## 方案概览

| 项目 | 说明 |
|------|------|
| **`左 Ctrl` / `左 Alt`** | 交换（与 Mac 修饰键位置一致） |
| **`Caps Lock`** | 单击 → `Esc`；长按进入导航层（方向、翻页、词跳、选中、删除、撤销等） |

各平台具体键位表与快捷键见对应子目录 README。

## 平台与用法

| 平台 | 说明与用法 | 实现 |
|------|------------|------|
| **Linux** | [Linux/README.md](Linux/README.md) | [keyd](https://github.com/rvaiya/keyd) |
| **Windows** | [Windows/README.md](Windows/README.md) | [AutoHotkey v2](https://www.autohotkey.com/) |
| **Mac** | [Mac/README.md](Mac/README.md) | [Karabiner-Elements](https://karabiner-elements.pqrs.org/) |

按需进入对应目录，按该目录 README 安装并启用即可。
